# Dietitian Website (Gediyet)

A modern nutrition consultancy website built for a professional dietitian.
The website presents services, consultation packages, client testimonials, and includes an interactive calorie calculator for visitors.

This project was developed to create an online presence for a nutrition expert and allow potential clients to easily learn about services and contact the dietitian.

Live website:
https://boran77.github.io/Dietititan_Website/

---

## Features

- Responsive landing page
- About section introducing the dietitian
- Services section (personalized nutrition, sports nutrition, functional nutrition)
- Consultation packages with pricing
- Client testimonials
- Interactive calorie needs calculator
- Contact form for potential clients
- **Working email system** that sends client inquiries directly to the dietitian's email

## Contact Form & Email Integration

The website includes a functional contact form that allows visitors to send consultation requests directly from the website.

When a user submits the form:
1. The message is processed by the form system
2. The inquiry is automatically sent to the dietitian's email
3. The dietitian can respond directly to the client

This enables potential clients to easily reach out for consultations without leaving the website.

## Website Sections

### Home

Introduces the nutrition center and highlights the goal of helping people improve their health through personalized nutrition programs.

### About

Provides background information about the dietitian and areas of expertise.

### Services

Lists the main services offered:

* Personalized nutrition plans
* Sports nutrition
* Functional nutrition

### Packages

Displays different consultation packages with pricing and included services.

### Calorie Calculator

Interactive tool that allows users to estimate their daily calorie needs based on:

* gender
* age
* height
* weight
* activity level
* fitness goal

### Testimonials

Client feedback showcasing successful weight loss and positive results.

### Contact

Allows visitors to send inquiries and request consultations.
